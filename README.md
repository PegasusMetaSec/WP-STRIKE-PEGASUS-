## 🚀 Tentang WP-STRIKE PEGASUS

**WP-STRIKE PEGASUS** adalah solusi all-in-one untuk keamanan dan optimalisasi WordPress. Dirancang untuk pengembang, agensi, dan pemilik situs yang menuntut performa tinggi dan perlindungan proaktif terhadap ancaman cyber.

> *"Flying faster than threats, striking harder than vulnerabilities."*

### ✨ Fitur Unggulan

| Modul | Deskripsi |
|-------|------------|
| 🛡️ **Pegasus Firewall** | Web Application Firewall (WAF) cerdas dengan deteksi anomali real-time. |
| ⚡ **Turbo Cache** | Caching dinamis + static HTML + Redis/Memcached support. |
| 🔍 **Malware Scanner** | Pemindaian otomatis file core, plugin, dan theme dengan signature update. |
| 🧹 **DB Cleaner** | Optimasi database, hapus revisi berlebih, spam, dan transien usang. |
| 🔐 **Login Armor** | Proteksi brute-force, 2FA, reCAPTCHA, dan whitelist IP. |
| 📊 **Audit Log** | Catat semua aktivitas admin & user secara detail. |
| 🌐 **CDN Integration** | Siap pakai dengan Cloudflare, BunnyCDN, dan StackPath. |

## 📦 Instalasi

### Persyaratan Sistem
- WordPress 6.0 atau lebih baru
- PHP 7.4 – 8.2
- MySQL 5.7+ / MariaDB 10.3+
- dukungan mod_rewrite (Apache) atau Nginx

### Langkah Instalasi

1. **Unduh** file ZIP dari rilis terbaru.
2. Masuk ke **Admin WordPress → Plugins → Tambah Baru → Unggah Plugin**.
3. **Aktifkan** plugin.
4. Buka menu **WP-STRIKE → Setup Wizard** untuk konfigurasi awal.

Atau instal melalui CLI:
```bash
wp plugin install wp-strike-pegasus.zip --activate
